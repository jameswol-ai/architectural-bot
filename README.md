# architectural-bot
Architectural AI Assistant
Architectural Feasibility & Costing Tool

Version 1.0 – Internal Use Only
Overview

This tool is a preliminary architectural feasibility assistant designed to support early-stage residential design decisions.

It generates conceptual layouts, checks simplified British Standards compliance, evaluates site constraints, and provides high-level cost estimates.

Intended Use
Early feasibility studies
Test-fit analysis
Budget guidance
Internal design discussions
Client briefing support (non-contractual)

Not Intended For
Planning submissions
Construction drawings
Legal or statutory compliance certification
Final cost tendering

Core Features
Multi-floor residential layout generation
Climate-aware zoning logic
British Standards–based compliance checks (simplified)
Room adjacency & privacy validation
Site orientation & setback analysis
UK cost estimation (£/m²)
BOQ-style cost breakdown
Room-by-room schedules
Downloadable text reports

Design Standards Referenced (Simplified)
UK Approved Documents (G, K, M, L – guidance level)
Typical UK residential best practices
Hot / temperate climate design logic
⚠️ This tool provides guidance only and does not replace professional judgment.

Cost Assumptions
Costs exclude land, VAT, professional fees, utilities
Rates reflect early-stage UK residential averages
Final costs subject to market conditions
